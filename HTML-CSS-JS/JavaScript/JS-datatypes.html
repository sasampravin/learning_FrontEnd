DataTypes in JavaScript

1> JS is dynamically typed lang
2> we use var to hold anytype of data type in JS ex--numbers,strings,etc.

Types of DataTypes in JS

JS Premitive DataTypes

a> string -- sequence of chars
b> number -- numeric values
c> boolen -- true or false
d> undifined -- undifined values
e> null -- null i.e. no value at all

JS non-premitive DataTypes

a> object -- instance through which we can accsss members.
b> array --- group of similar values
c> regexp -- regular expression
