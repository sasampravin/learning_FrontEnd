UnitsInCSS

1) pixcels ----- Same in maximum, minimum screen and not dependent on parent elements and text will be adjested accourdingly
2) em ---------- relative and dependent on parent elements and 1em=16px (default browser style)
3) rem --------- dependent on parent elements 1rem=16px (default browser style)
Note: If we give values in percentages it is dependent on parent elements but pixcels values or not dependent on parent elements.
4) vw ---------- ViewWidth, it(values) will occupying the percentages of the screen.
5) vh ---------- ViewHeight, it(values) will occupying the percentages of the screen.
